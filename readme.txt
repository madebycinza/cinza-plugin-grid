A minimal grid plugin built with Isotope.

== Description ==
Cinza Grid is a minimal grid plugin built with <a href="https://isotope.metafizzy.co/" targe="_blank">Isotope by Metafizzy</a>. It makes lightweight and flexible grids that can be easily customized with CSS.

== Installation ==
Upload the Cinza Grid plugin to your WordPress site, activate it, and start creating grids.
